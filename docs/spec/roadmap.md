# Roadmap

## Milestone 0: Scaffold (current)
- Package metadata and Apache-2.0 licensing.
- MVP plugin skeleton.
- Baseline tests and planning docs.

## Milestone 1: MVP release
- Stabilize config and command behavior.
- Add integration tests with sample MkDocs project.
- Add integration fixture checks for `pillar3` and `wp-sdmx-dpm` style projects.
- Publish alpha to PyPI.

## Milestone 2: Quality and usability
- Better table/image handling guidance.
- Optional TOC and chapter split controls.
- Improved logging and dry-run mode.
- Optional output split by top-level nav section.
- Option to include only nav pages (exclude non-nav docs files).

## Milestone 3: Advanced exporters
- Alternative PDF engine path (Playwright/Chromium).
- Per-page and multi-output strategies.
- Template packs/examples.
- Mermaid cache and render acceleration strategies.
